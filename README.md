# 实验版
# 特色
此版本仅将网上已有规则进行转换，以实现脱离外部订阅转换服务，直接使用机场原始订阅链接的情况下，仍能保持分流规则的一致性、体验的连续性。
目的是验证设想的可行性以及学习全局覆写配置功能的有关用法。推荐切换至其它分支，使用其它版本。
# 使用方法
